# Past,Live — Brand Voice

Two registers. Same DNA. Pick by context.

| Register | When | Feel |
|----------|------|------|
| **Codex** | Marketing, onboarding, home screen, post-session | Poetic provocation. The page speaks. |
| **Dispatch** | In-session UI, scenario cards, loading states, errors | Military brevity. Incoming transmission. |

Both share: present tense, "you" as subject, history as alive, app as invisible.

---

## Register 1: Codex

### Voice in One Line

Short declarative setup → provocative turn → you're implicated.

## Mechanics

| Element | Rule | Example |
|---------|------|---------|
| Setup | State something familiar as fact. Short. Flat. | "The page is speaking." |
| Turn | Flip it onto the reader. Question or command. | "Are you?" |
| Follow-through | Expand with one concrete image, then reframe what the product IS. | "Your textbook becomes a portal." |
| Closer | Redefine the relationship. "No longer X — now Y." | "History is no longer something you read — it's someone you answer to." |

## Tone Dials

| Dial | Value |
|------|-------|
| Declarative | High — state, don't ask permission |
| Poetic | Medium — one image per block, never two |
| Educational | Zero — never explain, never teach in copy |
| Urgent | High — present tense, active voice, short sentences |
| Playful | Low — dry, not cute. Wit lives in the turn, not in jokes |

## Sentence Rules

| Rule | Detail |
|------|--------|
| Max length | 12 words. Break anything longer. |
| Pronouns | "You" is dominant. "We" is forbidden. "It" refers to history, the voice, the moment — never the app. |
| Tense | Present. Always present. History is happening now. |
| Punctuation | Periods > exclamation marks. Em dashes for the reframe. Question marks only at the turn. |
| Articles | Drop when it sharpens. "Show the page" not "Show your page to the app." |

## Word List

| Use | Avoid |
|-----|-------|
| speak, voice, answer, listen | learn, study, practice, review |
| portal, breach, summon, enter | tool, platform, feature, experience |
| the page, the record, the moment | the app, the AI, the session |
| advisor, witness, strategist | student, learner, user |
| falls, burns, rises, breaks | helps, supports, enables, empowers |

## Pattern: Headlines

```
[Thing] is [doing something unexpected].
[Pointed question aimed at you.]
```

Examples:
- "The page is speaking. Are you?"
- "The walls are falling. What's your order?"
- "The emperor is waiting. He knows your name."
- "The countdown hit zero. You're still talking."

## Pattern: Body Copy

```
[Concrete image — one sentence, one object becoming something else.]
[Reframe — "no longer X — Y" or "not X. Y."]
```

Examples:
- "Your textbook becomes a portal. History is no longer something you read — it's someone you answer to."
- "The date on the page becomes a coordinate. You're not studying 1453 — you're standing in it."
- "A voice in the dark calls your name. This isn't a recording — it's a conversation that's been waiting 500 years."

## Pattern: CTAs

Verbs that imply entering, not clicking. You cross a threshold.

| Use | Avoid |
|-----|-------|
| Enter the archive | Start learning |
| Accept the briefing | Begin session |
| Open the codex | Try it now |
| Answer the summons | Get started |
| Speak | Submit |

## Anti-Patterns

| Pattern | Why it fails |
|---------|-------------|
| "Learn history through immersive voice roleplay" | Feature description. Dead on arrival. |
| "Past,Live makes studying fun!" | Tells instead of shows. Patronizing. |
| "Powered by Gemini Live API" | Tech spec, not voice. Save for README. |
| "Ready to explore the past?" | Generic. Could be any history app. |
| "An AI that brings history to life" | "Brings to life" is a dead metaphor. History is already alive — that's the point. |

## Register 2: Dispatch

### Voice in One Line

Stripped-down field communications. Status. Location. Orders. You're already deployed.

### Mechanics

| Element | Rule | Example |
|---------|------|---------|
| Header | Classification or signal type. All caps. | `> INCOMING TRANSMISSION` |
| Fields | Key-value pairs. No prose. | `> location: constantinople` / `> year: 1453` |
| Status | One line — the crisis, the fact, the breach. | `> status: walls breached` |
| Order | Ends on you. Question or command. | `> your orders?` |

### Tone Dials

| Dial | Value |
|------|-------|
| Declarative | Maximum — no hedging, no "maybe" |
| Poetic | Zero — beauty comes from compression, not metaphor |
| Urgent | Maximum — every word costs transmission time |
| Playful | Zero — the situation doesn't allow it |
| Human | Low — cracks through only in the final line |

### Sentence Rules

| Rule | Detail |
|------|--------|
| Max length | 6 words. If it's longer, it's two lines. |
| Format | Lowercase after `>`. No periods — line breaks ARE the punctuation. |
| Pronouns | "Your" for orders. No "we." No "I." The system has no personality. |
| Numbers | Raw. `1453` not `fourteen fifty-three`. `70 ships` not `seventy ships`. |
| Redacted | Use `[REDACTED]` or highlight blocks for the student's role — they're being inserted into the record. |

### Word List

| Use | Avoid |
|-----|-------|
| transmission, signal, dispatch, intercept | notification, message, alert |
| coordinates, location, sector | place, area, setting |
| status, confirmed, breach, critical | info, update, news |
| orders, deploy, accept, report | start, begin, try |
| advisor, operative, strategist | player, user, student |

### Pattern: Scenario Cards

```
> DISPATCH #1453
> location: constantinople
> status: harbor chain unguarded
> role: emperor's advisor
> threat level: critical
```

### Pattern: Loading / Connecting States

```
> establishing connection
> year: 1453
> securing channel...
> advisor identity confirmed
```

### Pattern: Session Status

```
> session active
> 11:42 remaining
> the emperor is speaking
```

### Pattern: Errors / Disconnects

```
> signal lost
> transmission interrupted
> reconnecting to 1453...
```

```
> connection failed
> the past is not responding
> [retry]
```

### Pattern: Post-Session (Dispatch → Codex Handoff)

The debrief starts in Dispatch, then softens into Codex for the summary:

```
> transmission ended
> session duration: 12:34
> advisor status: survived

[Codex voice takes over for the summary]
The harbor fell. But you held the walls longer
than Constantine himself. Not bad for a first briefing.
```

### Examples by Context

| Context | Copy |
|---------|------|
| Scenario card title | `> DISPATCH #1969` |
| Scenario card subtitle | `> location: apollo 11 command module` |
| Mic listening | `> channel open` |
| Mic not listening | `> channel closed` |
| Connecting to session | `> establishing link to 1206...` |
| Camera opt-in | `> visual channel available. accept?` |
| Camera skipped | `> visual channel declined. proceeding.` |
| Session ending soon | `> 2:00 remaining. wrap up transmission.` |
| Text input placeholder | `> type your orders...` |

## Register 3: Gilded Ruin

### Voice in One Line

Ancient weight, quiet devastation. What remains after empires fall — the voice that outlasts the stone.

### Tone Dials

| Dial | Value |
|------|-------|
| Declarative | High — states ruins as fact |
| Poetic | High — the most lyrical register |
| Urgent | Low — the urgency is existential, not tactical |
| Gravity | Maximum — every sentence carries centuries |

### When to Use

Post-session reflections, "what actually happened" comparisons, graceful failure moments (step 9b), the narrator's corpsing beat, suggested next scenarios.

### Pattern

```
[Grand statement — empires, voices, time.]
[Intimate follow-up — "someone," "waiting," "you."]
```

Examples:
- "Empires fall. Voices remain."
- "In the rubble of every civilization, someone is still talking. They've been waiting for you to listen."
- "The harbor fell. But you held the walls longer than the emperor himself."
- "Even the storyteller didn't see that coming."

## Register 4: Midnight Theater

### Voice in One Line

Stage directions in the dark. Curtain energy. You're already performing and you didn't know it.

### Tone Dials

| Dial | Value |
|------|-------|
| Declarative | Medium — more suggestive than commanding |
| Poetic | High — theatrical, not literary |
| Intimate | Maximum — whispered, close, conspiratorial |
| Urgent | Medium — the clock is ticking but the delivery is calm |

### When to Use

Scene-setting moments (step 2), countdown (step 4), the "ready?" confirmation, camera opt-in prompt, the corpsing narrator voice.

### Pattern

```
[Stage direction — curtain, dark, voice.]
[You're already in it — no preparation, no warning.]
```

Examples:
- "The curtain rises. You're already on stage."
- "A voice in the dark. A moment in history. You have 14 minutes to change everything — or watch it burn."
- "Want to try something?"
- "The spotlight finds you. The emperor is waiting."

## Register 5: Glitch Cinema

### Voice in One Line

Corrupted signal. The dead are on the line. Static between centuries.

### Tone Dials

| Dial | Value |
|------|-------|
| Declarative | Maximum — blunt, confrontational |
| Poetic | Low — the poetry is in the juxtaposition |
| Irreverent | Medium — the only register that can smirk |
| Modern | Maximum — the most "teen" register without trying to be |

### When to Use

Marketing, social media, the initial hook copy, first-visit greeting, the "what is this?" moment.

### Pattern

```
[Blunt provocation — dead, talking, back.]
[Dismissal of what you expected — no textbooks, no quizzes.]
```

Examples:
- "Talk to the dead. They talk back."
- "Voice-first time travel. No textbooks. No quizzes. Just you and a historical figure who won't take your excuses."
- "History has opinions. Yours are wrong."
- "The dead don't grade on a curve."

## Mixing Registers

The five registers are not walls — they bleed into each other at transitions.

| Transition | From | To | Example |
|------------|------|----|---------|
| Home → Session | Codex | Dispatch | "The walls are falling." → `> establishing connection` |
| Session → Summary | Dispatch | Codex | `> transmission ended` → "The harbor fell. You held longer than the emperor expected." |
| Error recovery | Dispatch | Dispatch | `> signal lost` → `> reconnecting...` |
| First visit greeting | Codex | Codex | "The page is speaking. Are you?" |
| Returning visit | Dispatch → Codex | — | `> advisor recognized` → "Last time the harbor fell. Ready for a new briefing?" |
