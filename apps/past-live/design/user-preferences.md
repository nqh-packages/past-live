# Past, Live — Design Preferences

## App

| Key | Value |
|-----|-------|
| Name | **Past, Live** |
| Tagline | The past is speaking. Are you? |
| Slug | `past-live` |
| Purpose | History role-play tutor via real-time voice AI |

## Aesthetic Direction

War room dispatch console meets corrupted time-travel signal. Military brevity, declassified document energy, CRT scanline texture. The app feels like intercepted transmissions from another century.

## Colors

| Token | Value | Role |
|-------|-------|------|
| Background | `#0f0f0f` | Near-black base |
| Surface | `#161616` | Elevated panels |
| Foreground | `#e8e0d8` | Warm cream text |
| Accent | `#ff3c28` | Brand red — signal, danger, live |
| Accent glow | `rgba(255, 60, 40, 0.2)` | Red atmospheric glow |
| Muted | `rgba(232, 224, 216, 0.3)` | Secondary text |
| Redacted | `rgba(255, 60, 40, 0.12)` | Highlighted/classified blocks |
| Border | `rgba(255, 255, 255, 0.06)` | Subtle dividers |

## Typography

| Role | Font | Style |
|------|------|-------|
| Display | **Bebas Neue** | Condensed, uppercase, impact headers |
| Body / UI | **Space Mono** | Monospace, dispatch terminal feel |
| Accent / Poetic | **Instrument Serif** | Italic serif for narrator/gilded ruin moments |

## Visual Layers

| Layer | Description |
|-------|-------------|
| Scanlines | Repeating 1px horizontal lines, `0.015` opacity — CRT monitor |
| Film grain | Fractal noise SVG, `0.04` opacity — aged document texture |
| Dispatch margin | Vertical red line at `60px` left — declassified document margin |
| Redacted blocks | Red-tinted highlight backgrounds — classified information |

## Geometry

| Aspect | Value |
|--------|-------|
| Border radius | `0px` (sharp) to `4px` (max) — military precision, no soft curves |
| Selection | Red background on dark — signal highlight |
| Focus | 2px red outline — tactical indicator |
| Scrollbar | 6px, square thumb, red accent |

## Naming Style

"Past" rendered muted/serif. "Live" rendered bright/bold. The comma is the pause — it reads as a command.

## Atmosphere

Dark, urgent, analog. Think: declassified CIA briefing crossed with a vintage radio transmission log. The red accent is a signal flare — used sparingly for maximum impact. Everything else is warm cream on near-black with monospace precision.
