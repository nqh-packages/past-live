## Critical Missing Feature: Scene Image Display

The backend sends `{ type: 'scene_image', title: string, image: string }` via WebSocket when the `show_scene` tool fires. The client WebSocket handler at `src/lib/liveSession/client.ts:114-218` has NO case for `scene_image`. Images are generated but never shown to the student.

### Protocol (from `server/src/protocol.ts:40`)

```typescript
| { type: 'scene_image'; title: string; image: string }
```

- `title`: Short title like "The ships being dragged over the hills"
- `image`: Base64-encoded PNG image data
